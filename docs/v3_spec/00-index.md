# Zutai Language Specification v3 — Deferred Features

v3 builds on the v0 core language, the v1 deferred feature set, and the v2
deferral horizon. This directory tracks language features that need design work
after v2 rather than standard-library API additions.

Current deferred design topics:

- [Generator and yield syntax](01-generators.md) — syntax and semantics for
  stream-producing language forms.
